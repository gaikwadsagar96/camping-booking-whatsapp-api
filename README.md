# Camping Booking WhatsApp API Utility

This is a simple utility concept for handling booking confirmations via WhatsApp.

## How it works
This logic helps automate customer messages after a successful database entry.
It is currently being used in production for the booking system at:
[Camping At Pawana](https://campingatpawana.com/)

## Features
* Connects to Fast2SMS API
* Sends confirmation on successful booking
* Handles user data security
